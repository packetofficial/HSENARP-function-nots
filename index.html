FUNCTIONS / METHODS NOTES

DEFINITION OF FUNCTION -

A function is a named block of instructions. Statements in a function can be executed several times using the name of the function.

ADVANTAGES OF DIVIDING A PROGRAM INTO DIFFERENT FUNCTION


Dividing a program into different functions help us to

1.Reduce the number of lines in a program.

2.Reduce the complexity of solving a problem

FORMAT OF DEFINING A FUNCTION


* In the syntax the square brackets indicate optional component


return type: The return type is the data type of the value themethod returns.If a method does not return any value then the return type is the keyword void .


Method Name: This is the name of the method given by the programmer. Any valid identifier can be used as the method name .


Parameter list: The parameter list refers to the data type, order, and number of the parameters of a method. Parameters are optional; that is, a method may not contain any parameter. Parameters areused in a method to receive values for processing.


Format -

(data type var1, data type var2,......,data type varn)

(int a,b,c)-> Wrong format of declaring variables in the parameter list

Method Body: The method body contains a collection of statements that define what the method does.

return :

-Keyword return is used to transfer a value from one function to another function.

-It is also used to terminate a function

Note: A function can return only one value even though several return statements can be written inside a function.

Function header/function prototype:


The first line of defining a function that contains the return type function name and parameter list is called function prototype

QUESTION 1.Write the prototype of a function named testthat takes two integers and a character as parameters and returns a logical constant.

ANSWER- boolean test(int a, int b, char ch)

QUESTION 2.Write the prototype of a function namedinterestthat takes principle, rate and time (double values) as parameters and returns simple interest.

ANSWER- double interest(double p, double r, double t)

QUESTION 3.Write the prototype of a function named processthat take variables of four different integer numeric data types as parameters and returns the sum of the parameters.

ANSWER- long process (byte b, short s, int i, long l)

QUESTION 4.Write the prototype of a function named area that takes the length and the breadth (double values) of a rectangle as parameters and displays the area.

ANSWER- void area(double l, double b)
